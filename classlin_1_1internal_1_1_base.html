Detailed Description

template<class D>
class lin::internal::Base< D >

Value backed tensor interface with resizing support.

Template Parameters
DDerived type.

This is the third of the three main entry points on the lin inheritance tree after internal::Mapping.

Here, direct tracking of runtime dimensions and the ability to resize is added. A getter to retrive a pointer to the element backing array is also included.

The main purpose is to provide an interface that supports value backed types

  • i.e. those directly storing tensor elements in a member array or and a pointer.
